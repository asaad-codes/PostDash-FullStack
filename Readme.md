# 🚀 PostDash - Full-Stack Image Sharing Platform

A high-performance MERN stack application focused on robust backend architecture, seamless cloud integration, and a modern user experience.

## 🧠 Backend Focus (My Contribution)
As a **Backend Developer**, I architected the server-side logic to handle complex media uploads and data persistence:

- **Image Processing:** Implemented a secure pipeline to handle image buffers and stream them directly to **ImageKit.io**.
- **RESTful API:** Developed scalable endpoints for post creation and data retrieval using Node.js and Express.
- **Database Architecture:** Optimized MongoDB schemas for efficient storage and fast query performance.
- **Middleware Integration:** Used `Multer` for handling multipart/form-data and memory storage for optimized performance.

## 🛠️ Technical Tech Stack
- **Server:** Node.js, Express.js
- **Database:** MongoDB (Mongoose ODM)
- **Media Hosting:** ImageKit Cloud API
- **Frontend Integration:** React.js, Tailwind CSS (Used for UI display)
- **Icons & UI:** Lucide-React

## ✨ Key Features
- ✅ **Secure Image Uploads:** Handled via server-side ImageKit integration.
- ✅ **Dynamic Feed:** Real-time data fetching from MongoDB.
- ✅ **Responsive Design:** A clean, masonry grid layout for better content visualization.
- ✅ **Error Handling:** Robust backend validation and error response management.
